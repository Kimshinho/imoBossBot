# 🤖 아이모 보스탐 매니저

Discord 봇으로 아이모(iMo) 게임의 보스 타이머를 관리하는 자동화 시스템입니다.

## 📋 기능

### 기본 기능
- 🎮 보스 등록 및 관리
- ⏱️ 자동 젠 타이머 계산
- 📢 타임 알림 시스템 (5분, 3분, 1분 전 알림)
- 📊 현재 보스 현황 조회
- ✅ 보스 컷 기록

### 최신 추가 기능 (v1.1)
- ✨ **보스 등록** - 커스텀 보스 추가 기능
  - 명령어: `보스등록 02:30:00 여왕(ㅇㅇ)`
  - 새로운 보스를 동적으로 등록 가능
  - 시간, 이름, 약어 자동 검증

- 🗑️ **보스목록 삭제** - 보스 완전 삭제 기능
  - 명령어: `보스삭제 깬`
  - 보스목록에서 영구 삭제
  - 기본 보스는 자동으로 보호

## 📖 사용 방법

### 1. 봇 초기화
```
!쿠이
```
사용할 채널 설정

### 2. 보스 목록 확인
```
보스목록
```
전체 등록된 보스 조회

### 3. 보스 등록 ⭐ NEW
```
보스등록 02:30:00 여왕(ㅇㅇ)
```
형식: `보스등록 HH:MM:SS 보스이름(약어)`

### 4. 보스 컷 기록
```
와당컷
```
또는 초성으로 `ㅇㄷㅋ`

### 5. 과거 시간 입력
```
1304 깬컷
```
또는 `깬컷 13:04`

### 6. 보스 삭제 (현황에서만 제거)
```
깬삭제
```

### 7. 보스목록 삭제 ⭐ NEW
```
보스삭제 깬
```
보스목록에서 완전 삭제

### 8. 현황 조회
```
보스탐
```
또는 `현황`

### 9. 메뉴얼 확인
```
메뉴얼
```

## 🛠️ 기술 스택

- **언어**: Python 3.13
- **라이브러리**: discord.py, python-dotenv
- **저장소**: JSON 파일 기반 데이터 관리

## 📁 프로젝트 구조

```
imo-boss-bot/
├── main.py              # 봇 진입점
├── core/
│   ├── bot.py          # 봇 설정
│   ├── boss_data.py    # 보스 데이터 관리
│   ├── config.py       # 설정
│   └── storage.py      # 메모리 저장소
├── events/
│   ├── message.py      # 메시지 이벤트 처리 ⭐ 변경됨
│   ├── ready.py        # 봇 준비 완료 이벤트
│   └── errors.py       # 에러 처리
├── tasks/
│   └── boss_loop.py    # 보스 타이머 루프
├── utils/
│   ├── message_sender.py
│   ├── parser.py
│   ├── sender.py
│   └── time_utils.py
├── views/
│   └── move_confirm.py # 채널 이동 확인
├── data/
│   ├── boss_state.json
│   └── custom_bosses.json    # ⭐ 커스텀 보스 저장소
├── requirements.txt
└── README.md           # 이 파일
```

## 🔧 환경 설정

### 필수 패키지 설치
```bash
pip install -r requirements.txt
```

### Discord Bot Token 설정
`.env` 파일에 다음과 같이 설정:
```
DISCORD_TOKEN=your_bot_token_here
BOSS_CHANNEL_ID=your_channel_id
```

## 🚀 실행 방법

```bash
python main.py
```
